# comment-clustering-ai

コメント（テキスト）を埋め込みベクトル化し、クラスタリングしてテーマ別に整理するためのプロジェクトです。

## ディレクトリ構成

| パス | 説明 |
|------|------|
| `docs/` | 設計メモ・仕様などのドキュメント |
| `src/` | ソースコード |
| `data/` | 入力データ（CSV など）を置く |
| `output/` | クラスタリング結果の出力先 |

## セットアップ

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```
